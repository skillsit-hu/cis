# CIS Docker
## Setup
```sh
git clone https://github.com/skillsit-hu/cis.git
cd cis
docker compose up -d
```
#### Admin User and Password
``admin:admin``

### Apps
- CIS: [http://localhost:9090](http://localhost:9090)
- PMA: [http://localhost:8080](http://localhost:8080)
- DB: [http://localhost:3306](http://localhost:8080)
